# 🚀 ResumeForge AI

## 📌 Inspiration
Many students struggle with creating a professional resume and don’t know how to improve an existing one. Even when they have a resume, they are often confused about what to add, remove, or enhance. This inspired us to build a solution that not only creates resumes but also intelligently guides users to improve them.

---

## 💡 What It Does
ResumeForge AI is a **Generative AI-based Resume Builder and Enhancer**.

It allows users to:
- Create a new resume from scratch  
- Upload an existing resume  
- Get AI-powered suggestions to improve:
  - Content  
  - Structure  
  - Keywords  

The system helps users make their resumes more professional and ATS-friendly.

---

## 🛠️ How We Built It
- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **AI Integration:** Generative AI for resume analysis and suggestions  

The frontend provides a smooth and user-friendly interface, while the backend handles processing and AI-driven insights.
